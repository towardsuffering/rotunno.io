## Project Overview

The marketing team was running budget decisions off a last-touch attribution model baked into their ad platform. Analysis showed the model was systematically undervaluing upper-funnel channels (content, organic search) and overvaluing retargeting — leading to an estimated $400K/year in misallocated spend.

I built a multi-touch attribution pipeline from scratch, connecting raw ad event data to CRM-sourced closed revenue.

## Technical Implementation

Source data: Google Ads, Meta, LinkedIn, HubSpot, and Salesforce — all landed in BigQuery via Airbyte. I built a Python script implementing a time-decay attribution model with configurable channel weights, wrapped in a dbt Python model for auditability. The attribution output fed into a Looker dashboard with a side-by-side comparison view showing last-touch vs. multi-touch attribution by channel, campaign, and time period.

The hardest problem was identity resolution — matching ad impressions and clicks to CRM contacts across platforms with different identifiers. I built a deterministic matching layer using email hash + device fingerprint, with a probabilistic fallback.

## Outcome

Marketing adopted the multi-touch model for Q3 planning. Budget reallocation toward upper-funnel channels resulted in a 22% improvement in cost-per-acquisition over the following two quarters. The CMO presented the attribution analysis at the annual board meeting.
