Great data teams build knowledge, not just dashboards. I offer structured training and mentorship for analysts growing into engineering roles and for teams adopting a modern data stack for the first time.

## Topics Covered

- Advanced SQL patterns (window functions, CTEs, incremental design)
- Python fundamentals for data work (pandas, SQLAlchemy)
- Data modeling concepts (dimensional modeling, OBTs, metrics layers)
- Dashboard design principles and stakeholder communication
- Git workflow for data teams

## Format

1:1 mentorship · Team workshops · Async code review · Documentation review

I teach the way I'd want to be taught — with real examples, clear tradeoffs, and enough context to make independent decisions.
