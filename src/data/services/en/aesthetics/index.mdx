Clean data doesn't arrive — it's engineered. I design and build the pipelines that move data from source systems into your warehouse reliably, with tested transformations your team can trust and maintain.

## What's Included

- Source system integration and ingestion design
- ELT pipeline implementation (Fivetran, Airbyte, custom Python)
- dbt transformation layer — modeled, tested, and documented
- Orchestration with Airflow or dbt Cloud
- Incremental model design for production scale

## Stack Experience

I write transformation code the way software engineers write application code — version-controlled, reviewed, tested, and documented.
